# Useful adminstrative functions

<Callout type={'info'}>
Full list of endpoints can be found in [Yaci Provider](https://meshjs.dev/providers/yaci).
</Callout>

Yaci Provider provides a set of administrative functions to interact with the Yaci instance. Here are some of the functions that can be used to interact with the Yaci instance:

### [Address Topupinfo](https://meshjs.dev/providers/yaci#addressTopup)

You can topup ADA for any address. To topup ADA in your wallet, run the following command from devnet:

```
await yaciProvider.addressTopup(<address>, <amount>)
```

### [Get Devnet info](https://meshjs.dev/providers/yaci#getDevnetInfo)

Get information about the devnet.

```
await yaciProvider.getDevnetInfo()
```

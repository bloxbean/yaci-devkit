Transfer ADA (CIP-30 Flow)

CIP-30 Standard Flow: Build an unsigned transaction on the server, sign it with the wallet (CIP-30 signTx), then submit via the wallet (CIP-30 submitTx). Works with any CIP-30 wallet (Yaci DevKit, Eternl, Lace, etc.).
